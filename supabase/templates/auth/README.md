# Supabase Auth email templates

These HTML templates match the MomentoVino brand (WINE `#722F37`, BG `#F5EBE0`,
serif headlines, sans body). They use Go template variables (`{{ .Var }}`)
that Supabase exposes in each email type.

## Install

Supabase Dashboard → **Authentication → Email Templates**:

| Template          | File              |
|-------------------|-------------------|
| Confirm signup    | `confirmation.html`     |
| Invite user       | `invite.html`           |
| Magic Link        | `magic-link.html`       |
| Reset Password    | `recovery.html`         |
| Change Email      | `email-change.html`     |
| Reauthentication  | `reauthentication.html` |

Paste each file's HTML into the corresponding template editor and save.

## Subjects (set in the same screen)

- Confirm signup: `Welcome to MomentoVino — confirm your email`
- Invite: `You're invited to MomentoVino`
- Magic Link: `Your MomentoVino sign-in link`
- Reset Password: `Reset your MomentoVino password`
- Change Email: `Confirm your new MomentoVino email`
- Reauthentication: `Confirm it's you — MomentoVino`

## Notes on rendering

- Email clients strip `<link>` to Google Fonts; we declare the brand fonts
  via `@import` AND fall back to system serifs/sans. Most clients (Gmail, Apple
  Mail) will render the system fallback — that's fine.
- Layout is single-column, max-width 560px, fully fluid below that.
- The CTA button is a table-based row with inline padding so Outlook honors it.
- Plain-text alt is auto-generated by Supabase; no need to provide one.
